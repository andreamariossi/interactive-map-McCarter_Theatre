# Interactive Arts & Ideas Map

This project is an interactive map that showcases how **McCarter Theatre** and **Princeton University** connect across campus through programs, partnerships, and stories. Each pin on the map opens a popup with an exciting contribution, including testimonials, photos, videos, and links to further resources.

**Live site:**  
https://andreamariossi.github.io/interactive-map-McCarter_Theatre/

This project is part of the **GradFUTURES** initiative.
